# Security Policy

## Supported Versions

Currently, only the latest release of the Cat Identifier AI pipeline and mobile application is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for this edge-deployment architecture. If you discover a security vulnerability within the model pipeline or the React Native application, please DO NOT open a public issue.

Instead, please email the repository owner directly. You will receive a response within 48 hours acknowledging receipt of the vulnerability and outlining the timeline for a patch.
